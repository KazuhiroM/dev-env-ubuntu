Name
====
開発環境@ubuntu

Overview
====
いろんな開発を行うための生ubuntu環境

Description
====
- ubuntu 22.04 

Requirement
====
- Docker Desktop
- Docker compose

Install
====
```
cd [docker-composeファイルの場所]
docker compose up -d --rebuild
```
Usage
====
```
docker compose exec vm-ubuntu bash
```
